  
  RVTDX9VB 1.0 BETA RELEASE

  rvtDX9vb.tlb and rvtDX9vb.dll are together needed to allow use of Direct3D9 from VB6.

  Use the MakeDLL program to unpack the DLL and then copy to the right place.


  For the Whole thing to work you need to have installed

   DirectX 9.0c (Feb2005) from Microsoft. It will not work for any other version of DirectX 

   Make your VB program reference this library. Also read the .Doc.
